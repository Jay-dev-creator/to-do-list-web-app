About

The To-Do app is an app that keeps record of a list of tasks the user wants to complete. The app allows the user to delete, and also update the list, tasks completed can be scratched out using the 'complete' functionality that is included in the app.
The app was build by Kgatle JM using Django and deployed using heroku.
